# WcClaw

WcClaw (文出龙虾) 本地自动化桌面工具。

## 功能
- 本地模型管理（下载一键安装、启动/停止、自检）
- 自然语言任务生成与执行
- 任务引擎、日志与历史
- IM 管理（QQ/OneBot、Webhook；Telegram/Discord 可选高级）
- Skills 管理
- 自动更新（官网 update.json + GitHub Release）

## 发布包
请使用 `release/WcClaw-0.1.2-win64.zip`。

注意：如需本地模型运行，发布包中应包含 `tools/llama-server.exe`。

## 更新清单
`update.json` 用于客户端检查更新。
